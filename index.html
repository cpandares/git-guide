  • Git init inizializa git en nuestro repositorio: Solo se ejecuta una vez por directorio
  • Git add . (Toma todos los cambios en nuetro directorio y los prepara para el commit)
  • Git commit ( Toma los cambios y los pasa a un stage temporal antes de subir al repo, se agrega -m "Un mensage" ),
    git commit -m "mensaje"
  • Git log muestra el historial de commits
  • Git diif, nos muestra la diferecia entre 2 commits, git diff numerodecommit numerodeotrocommit
